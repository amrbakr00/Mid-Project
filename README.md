# Fraud Detection Analysis and Dashboard

## Overview
This project analyzes transaction data to detect fraudulent patterns. It includes data cleaning, exploratory data analysis, risk scoring, and an interactive Streamlit dashboard.

## Features
- Data cleaning and preprocessing
- Exploratory data analysis with visualizations
- Risk scoring model for transactions
- Interactive Streamlit dashboard
- Actionable recommendations

## Dataset
- 25,000+ transactions
- Features: transaction amount, time, device, location, payment method, previous fraud history, etc.
